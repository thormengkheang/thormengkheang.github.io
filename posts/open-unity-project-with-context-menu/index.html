Open Unity Project With Context Menu

November 29, 2015

Are you tired of having to open Unity3D to open your Unity3D project or sometimes you have to open Unity3D and press Open and search for your Unity3D project. Well no more, I found this awesome trick that would make opening Unity3D project way for more easier on both Windows and OS X.

First off, On Windows this is from Adam Tuliper an Developer Evangelist for Microsoft blog. It’s just registry file. You can either copy and paste it into your text editor of choice or you download it off this gist.

Note if you copy and paste from the gist, save it and give it a name but it must have an extension of .reg for example yourawesomename.reg.

After you saved it, you just double click on the .reg file you will be good to go.

Finally for OS X, Open Automator and select Services.

After that choose the dropdown item.

Note if you want to right click service to work in any apps, you can change the dropdown item Finder.app to any application.

Then you have to add Get Selected Finder Items action to the service.

Finally step is to add Run Shell Script action.

The very important thing to do now is to change Pass input: to stdin to Pass input: as arguments and remove the all the text and add script command below

/Applications/Unity/Unity.app/Contents/MacOS/Unity -projectPath "$1"

Note the -projectPath option is an command line option take from Unity Command Line Manual and the $1 is the first argument that are passed from the Get Selected Finder Items action.

After that you save the Automator workflow by pressing cmd+s and give it a name to whatever you want. The name of the workflow will be the name of right right click item. In the image below I saved my workflow as Open As Unity Project so the right click item will appear as Open As Unity Project.

From now on, you can open an Unity project from your right click.
